Este sitio fue hecho con inspiracion en el anime de Serial Experiments Lain y se uso dos maquinas para crear las dos versiones de la pagina, una HTML4/5 con una i5 y 16 gigas de RAM y una 486 a 66 MHz y 16 megas de RAM para la version HTML3/3.1
Las aplicaciones incluidas en la carpeta de descargas funcionaran con ambas versiones pero cabe destacar que el Netscape 3.0.5 no tiene soporte para codigo CSS.
